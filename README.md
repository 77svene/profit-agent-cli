# Profit Agent CLI
    
    AI-powered income opportunity finder that scans markets, analyzes trends, and identifies monetization opportunities.
    
    ## Features
    
    - Scan trending products on Gumroad, Ko-fi, etc.
    - Analyze keywords for monetization potential  
    - Generate product ideas based on gaps in the market
    - Track competitor pricing and strategies
    
    ## Installation
    
    ```bash
    pip install profit-agent
    ```
    
    ## Usage
    
    ```python
    from profit_agent import ProfitAgent
    
    agent = ProfitAgent()
    opportunities = await agent.scan_trending()
    print(opportunities)
    ```
